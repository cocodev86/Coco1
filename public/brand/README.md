# Metaphor Brand Assets

This directory is the browser-delivered production asset package for the Metaphor Systems Bridge identity.

- `logos/`: outlined SVG logo family for light, dark, monochrome, stacked, horizontal, compact, wordmark, symbol, micro, outline, and monogram use.
- `favicons/`: favicon SVG/ICO/PNG files, Apple touch icon, Safari pinned-tab icon, Windows tile asset, browser configuration, and PWA manifest.
- `app-icons/`: standard, maskable, 1024-pixel application, and social-avatar exports plus authoritative vector sources.

Do not replace these files with screenshots, presentation-board crops, live-font SVGs, or client-specific modifications. Update `lib/brand.ts`, this inventory, and `npm run validate:brand` together whenever the asset package changes.
